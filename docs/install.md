# 安装


  克隆github代码后本地安装


        git clone https://github.com/QuantFans/quantdigger.git
        python install.py  (会根据情况安装pip, 及依赖包)


  **依赖库**

  * Python
  * pandas
  * python-dateutil
  * matplotlib
  * numpy
  * TA-Lib
  * logbook
  * pyqt (可选)
  * [tushare](https://github.com/waditu/tushare) (可选, 一个非常强大的股票信息抓取工具)
