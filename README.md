
QuantDigger
=======
一个真在成长中的期货/股票交易框架。。。

安装
=======
 建议安装[Anaconda](http://continuum.io/downloads), 这样可以一次性搞定所有依赖包。

依赖库
=======
依赖库：
* Python 
* pandas 
* python-dateutil 
* matplotlib 
* numpy
* pyqt

运行
=======
* mplot_demo .py  matplotlib画k线，指标线的demo。
* pyquant.py 基于pyqt， 集成了ipython和matplotlib的demo。
